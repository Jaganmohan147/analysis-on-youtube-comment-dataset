About Project

Abstract

YouTube is one of the largest video platforms, and user comments provide valuable insights into audience opinions, trends, and engagement.
This project analyzes a YouTube Comments Dataset using Pandas Profiling to explore sentiment, spam detection, and comment trends. The
goal is to identify patterns in user interactions, common words, and sentiment polarity. 

Introduction

YouTube comments reflect user engagement, feedback, and reactions to videos. Some comments are positive, while others may contain spam or hate speech. This project aims to analyze comment length, sentiment, frequency, and common words to understand audience behavior.

Literature Survey

Studies show that YouTube comments can be categorized using Natural Language Processing (NLP) techniques. Researchers have used
sentiment analysis, spam detection, and word cloud visualizations to classify comments. Data analysis tools like Pandas Profiling help in
understanding patterns in large comment datasets.

EDA

EDA helps in analyzing audience engagement. Pandas Profiling is used to examine:
• Most frequent words – Identifying common topics in discussions.
• Sentiment analysis – Checking if comments are positive, negative, or neutral.
• Spam detection – Finding repetitive or promotional comments.
• Comment length distribution – Understanding if longer comments have higher engagement.
• Like & reply counts – Identifying the most engaging comments.
• Correlations – Finding relationships between sentiment, likes, and replies.
